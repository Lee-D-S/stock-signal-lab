# 주가 변동 원인 분석 정리

## 폴더 구조

- `00_기업별분석`: 기업별 보고서와 원본 분석 산출물
- `01_기획`: 프로젝트 목적, 실행 흐름, 해석 기준
- `02_기준`: 판단 기준과 실행 규칙
- `03_원천데이터`: 이벤트 집계와 분포 요약
- `04_패턴분석`: 패턴 후보와 국면별 분석
- `05_가설검토`: 가설별 이벤트 리뷰와 요약
- `06_백테스트`: 대리/실전 백테스트와 보조 결과
- `07_전략신호`: 전략 조건 초안과 관심종목 후보
- `08_관찰기록`: 확정 후보 관찰 로그
